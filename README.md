# Ассистент HiFi магазина
Это учебный проект из курса Prompt Engeneering от Zerocoder University.
Проект создает ассистента по записи на прослушивание аудиотехники через Telegram и виджет на веб-странице. Коммуникация с клиентом идет через OpenAI ассистента. Записи сохраняются в Google Sheets.

Приложение размещается в докере на VPS. Код создан с использованием ChatGPT.
